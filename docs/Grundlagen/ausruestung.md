# Ausrüstung

Ersetzt den externen „Ausrüstungsliste"-Pad
(`pad.gwdg.de/Kq3NVJ3tSS2pZuYQqhO1cg`, siehe `../quellen.md`). Begründung der
Stufung: Risk Compensation, siehe `../Konzept/verletzungspraevention.md`.

## Prinzip: leichte Ausrüstung früh
Mehr Schutz verleitet zu härterem Fechten. Daher beginnt der Kurs mit
**Schaum-/Nylonwaffen + leichtem Schutz** — das senkt die Intensität und fördert
technisches Fechten. Vollschutz ist **Folge** wachsender Kontrolle, nicht
Voraussetzung fürs erste Üben.

## Stufe 0 — Einstieg (erste Wochen, Anfänger)
| Teil | Empfehlung |
|---|---|
| Waffe | **Schaumschwert / Nylon-Langschwert** |
| Kopf | Fechtmaske (350N) sobald Partnerkontakt zum Kopf möglich |
| Hände | leichte Handschuhe |
| Kleidung | Sportkleidung, festes Schuhwerk |

## Stufe 1 — technisches Sparring (nach Trefferkontrolle)
| Teil | Empfehlung |
|---|---|
| Waffe | Nylon-Langschwert, später Feder (leicht) |
| Kopf | Fechtmaske 350N + **Hinterkopfschutz** |
| Hals | Halslatz / Gorget |
| Hände | **HEMA-Handschuhe** (Hauptverletzungszone Hand!) |
| Rumpf | Plastron / leichte Jacke |
| Ellbogen/Knie | leichte Protektoren |

## Stufe 2 — Vollkontakt-Sparring (Fortgeschrittene)
| Teil | Empfehlung |
|---|---|
| Waffe | Stahlfeder (HEMA-Standard) |
| Kopf | Maske + Overlay/Hinterkopf, **Fit-Check Pflicht** |
| Hände | schwere HEMA-Handschuhe |
| Rumpf | Fechtjacke (≥ 350N), Brust-/Rückenschutz |
| Gelenke | Ellbogen, Knie, Schienbein; Tiefschutz; Unterarmschutz |

## Erweiterungs-Waffen
- **Dolch:** Trainingsdolch (Nylon/Schaum), Maske, Handschutz.
- **Stange/Stab:** gepolsterte/Holz-Stange je nach Intensität, Maske, Handschutz.
- **Ringen:** Matte/weicher Untergrund, keine harten Protektoren.
- **Dussack/Säbel:** Nylon-Dussack/Säbel, Maske, Handschutz.
Details je Block in `../Erweiterungen/`.

## Beschaffung & Pflege
- Vereins-Leihbestand für Stufe 0/1, damit Einstieg ohne Eigenkauf möglich ist.
- Masken regelmäßig auf Gitterschäden prüfen; Handschuhe auf durchgescheuerte
  Stellen; Klingen auf Grate/Risse (siehe `sicherheit-kodex.md`, Waffencheck).
